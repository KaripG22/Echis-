 Echis 

> Μια Γλώσσα Προγραμματισμού στα Ελληνικά, βασισμένη στην Python.


Εγκατάσταση

1. Πηγαίνετε στη σελίδα [Releases](https://github.com/karipG22/Echis-/releases).
2. Κατεβάστε το τελευταίο αρχείο **`echis.exe`**.


! Τα αρχεία χρειάζονται κατάληξη .ech  π.χ. "try.ech"
και για να τρεξει: .\echis.exe try.ech !
