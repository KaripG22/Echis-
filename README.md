 # Echis 

 ## Μια Γλώσσα Προγραμματισμού στα Ελληνικά, βασισμένη στην Python.


### Εγκατάσταση

1. Πηγαίνετε στη σελίδα [Releases](https://github.com/karipG22/Echis-/releases).
2. Κατεβάστε το τελευταίο αρχείο **`echis.exe`**.


### UPDATE : echis --update


! Τα αρχεία χρειάζονται κατάληξη .ech  π.χ. "try.ech"
και για να τρεξει: .\echis.exe try.ech 
πλέον δέχεται και αρχεία .py !

Η Echis υποστηρίζει εντολές τόσο με τόνους όσο και χωρίς, καθώς και πεζά/κεφαλαία.

| Ελληνική Εντολή | Python Αντιστοιχία | Περιγραφή |
| :--- | :--- | :--- |
| **`εκτύπωσε(...)`** | `print(...)` | Εμφανίζει κείμενο στην οθόνη |
| **`διάβασε(...)`** | `input(...)` | Διαβάζει είσοδο από τον χρήστη |
| **`όρισε`** | `def` | Δημιουργία συνάρτησης |
| **`αν`** | `if` | Συνθήκη ελέγχου |
| **`αλλιώς`** | `else` | Το "αλλιώς" της συνθήκης |
| **`για` ... `σε`** | `for` ... `in` | Βρόχος επανάληψης (Loop) |
| **`όσο`** | `while` | Βρόχος όσο ισχύει συνθήκη |
| **`σπάσε`** | `break` | Σταματάει τον βρόχο |
| **`συνέχισε`** | `continue` | Συνεχίζει τον βρόχο |
| **`εισαγωγή`** | `import` | Εισαγωγή βιβλιοθήκης |
| **`κλάση`** | `class` | Ορισμός κλάσης (Αντικείμενα) |
| **`δοκίμασε` / `εκτός`** | `try` / `except` | Διαχείριση σφαλμάτων |
| **`γύρισε`** | `return` | Επιστροφή τιμής από συνάρτηση |
| **`ακέραιος(...)`** | `int(...)` | Μετατροπή σε ακέραιο |
| **`συμβολοσειρά(...)`** | `str(...)` | Μετατροπή σε κείμενο |

###  Τελεστές Λογικής
| Echis | Python |
| :--- | :--- |
| **`και`** | `and` |
| **`ή`** | `or` |
| **`όχι`** | `not` |
